# **Minimum Add to Make Parentheses Valid**

#### [_(Leetcode link for the problem)_](https://leetcode.com/problems/minimum-add-to-make-parentheses-valid/)

```java
class Solution {
    public int minAddToMakeValid(String S) {
        int open=0, ans=0;
        for(char i : S.toCharArray()){
            if(i=='(') open++;
            else {
                if(open==0) ans++;
                else open--;
            }
        }
        return ans+open;
    }
}
```
